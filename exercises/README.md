# Exercícios de Arduino

Esta pasta contém todos os exercícios práticos desenvolvidos para aprender Arduino, eletrónica básica e interação hardware/software.  

## Estrutura
- Cada exercício está em uma subpasta numerada, incluindo:
  - Código `.ino`
  - Pasta `media/` com imagens e diagramas
  - README.md do exercício (quando necessário)

## Lista de Exercícios
1. **01.basic_led_on** – Acender um LED continuamente
2. **02.led_blink** – Piscar um LED com temporização
3. **03.police_led** – Simular luzes de polícia com LEDs
4. **04.button_led** – Controlar LED com botão
5. **05.sensor_piezo_led** – Controlar LED com sensor piezo
